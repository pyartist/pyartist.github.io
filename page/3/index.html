半导体也在变革，ARM打破了原来的规矩

发表于 2014-10-16   |   分类于 未分类   |  

公司之间模块化开发，但还没有开源；台湾韩国的芯片设计和晶元场的压缩成本打破了原来的规矩

在这个蔬菜水果天价，数码产品白菜价的时代，低端的芯片在还没有中国集团的加入的情况下就已经从几美元到几人民币，到几毛了

我们现在的工作就如当年研发机器让机器化大生产代替手工业一样。将来对研发人员的需求将大大降低。更多的研发人员要转去提供服务。

就 如机器化大生产代替手工业并没有让社会的职位更少了，而是更多了，只是生活生产方式被深刻改变了。将来的网络社会让很多研发职位转到服务上去，应该也不是 职位变少了，而是变多了。因为随着城市化，随着人的需求增多，职位会更多，只是网络让工具让平台更智能更有效率更自动化而已。

未来是现实社会和网络社会并存，相辅相成的社会。

所谓技术革命就是为了让生产更有效率

未来肯定是“模块化开发；测试，系统管理自动化，工具化；开源”来代替“大公司垄断技术版权专利；人工开发，各个公司重复开发研究模块；低效率重复垄断技术保护竞争”。

大公司在技术革新面前，因为体积太大，让船转向非常慢和困难，这个时候，因为是网络时代，所以研发团队小型化，大型的研发可以通过开源社区，全球最顶尖的黑客们携手开发，版权可以通过GPL授权。

小 型团队，个人的力量凸显起来，在这个时代下可以实现，因为以前平台只能靠公司给，大公司给，现在有了网络平台，开源，工具，社区的发展和成熟。正如当年 8848没成功，淘宝成功了一样，8848在网络支付，物流，上网普及率没到一定程度的时候，自身再好也不会成功，相反，淘宝赶在了正确的时候，即使不是 淘宝不出现也会是另外一个类似的公司出现。

我记得大学里一个老师说过未来的社会的技术的发展是黑客引领的，当时不理解，也没当回事，现在明白了。这果然是个最好的时代，最公平的时代，最有效率的时代，最能发挥每个人无限潜力的时代。

放弃追究日本战争责任和战争赔偿的原因

发表于 2014-10-13   |   分类于 未分类   |  

当时的台湾，所谓的中华民国，放弃是因为迫于美国的压力，美国为了对付苏联和中国，强硬要求台湾和日本放弃相互对峙，加入并成为日本台湾旧金山纽约体系中的一个组成部分。

中 国放弃是因为想和日本建交，特别朝鲜战争后，美国第七舰队开进台湾，中国被封死，需要一个出口，就考虑日本，其实早在1950年的时候中国就考虑和日本建 立外交关系，还发展出了一套“一分为二”理论支持，就是现在大家还熟悉的把一部分军国主义分子和日本的广大人民分开来对待，责任也划开开了，这样一来，有 了这个前提条件才可以进行下去。中国为了表达这方面的意思，主动放弃了追究战争责任和战争赔偿。中国宣布放弃的同时中日共同宣布建交。

后 来者胡耀邦时代，日本发生以公职身份参拜靖国神社，中国强烈发对，日方完全没想到，和中国接触后得知，中国之所以让日方出乎意料的反对，是因为中国的“一 分为二”理论，中国已经把所有责任推到军国主义者身上，日本再以公职身份参拜就会动摇中国领导人的威信和执政地位，这个是不允许的，所以中国有这样的反 应。

日本一直认为中国的教育虽不能说是反日教育，但至少是仇恨教育。且自江泽民（偏左）上台后被利用成维稳，发泄不满因素出口。

第三方，就是西方认为，中日间可以通过民间交流改善，特别是中日友好医院，是很好的途径，既是民间的，又可以通过救人对当年的罪行做出弥补，且日本富裕和有先进的医疗，正好可以互补，帮助中国。

我觉得以上是对中日战争和责任比较中立的看法。

我个人的带有个人色彩的看法：
最理亏的还是台湾和日本，中国理亏成分最少。因为日本，和台湾在这件事情上没有自我意识，完全是跟随美国，亦步亦趋，是美国战略的一个棋子。相反，中国是有全局战略考虑的，一切战略都为了经济发展和维护稳定的周边环境。所以未来长远是中国会改变现在的格局的。

中日友好医院只能有一个两个，应该达不到西方希望的一千个，因为中国现在的医院民营的管制都很多，不要说外资了。

发表于 2014-10-11   |   分类于 未分类   |  

电视剧不是现实，不是教化，是另一个世界，可以用这个世界的眼光看观察另一个世界的事情，得到启发和感受

半泽的做法从逻辑上是行不通的，按半泽的做法，不可能能够事业平步青云，一直上升到最高层，然后被赶走的，早在初期就被拍死了。且同期的人应该能看出来他的做法有问题，应该早就从旁提点他

他有个好老婆，能够为了给老公换包，作为家庭主妇的她特地出去打工，为了用自己赚的钱为老公买一个新的包。自己本来可以凭借自己的专业能力在社会立足，但是为了家庭，把家庭放在第一位，放弃了自己这一块

两个大公司合并的大银行，有两派的人。高层一直有两派的人斗争。这样的 环境下，若不选择站一边，那么保持中立是不能混下去的

最 后行长把半泽调走，而只把犯错的自己另外一派的顶层的人降了一点级而已。之所以这么做，是因为有考虑的。后者是笼络另一派的人心，让另一派的头头永远不会 对自己构成威胁，若因为他犯错而调走他惩罚他，那么还有第二个他，第三个他，不如大事化小，赢得另一派对自己的臣服。把半泽调走是因为半泽只能是一个做事 的人，对维系高层组织没有任何用，而替代做事的人的人有很多。若让一个对高层没有帮助或者会添乱的人呆在高层，那是非常不明智的。行长是完美的赢家。

对于渡边初一的精神世界
渡边淳一:我首先要澄清一点:文学不是教化的手段,不要通过文学来教育我们去符合伦理,而是 要表达人的本性和真心。所以在文学作品中,经常会出现一些和我们的常识与伦理不相符的故事情节,这种做法是否对,是否符合伦理关系不是文学可以解决的。文 学应该是自由的,表达真情和本性,不是在论述伦理。另外,伦理观念会随着时间推移而变化,但是人性不会变,虽然它如此复杂。

有道理，我们老是把文学当教化，若有不好倾向引导，犯罪引导，就抵制，其实错误的。文学是对生活的提炼，比如将边界爱情（禁果），不是让人们去尝试边界爱情（禁果），而是让人们去感受，从而更好地理解现实世界，这才是艺术，艺术是高于生活的哈

我一直相信只有理性地控制自己感情的人，才是人生的赢家。
