在日本第十一天

发表于 2011-09-08   |   分类于 未分类   |  

在日本工作，觉得时间很是短暂，因为每天路上要花不少时间，且都要一板一眼吃早饭，吃一日三餐，要规规矩矩穿着，很累。总感觉休息睡眠比国内过，但觉得更累了。

日本和中国一样，并不是所有人都讲素质，走在日本的路上，时不时看到有人丢烟头，扔瓶子，随地吐痰。和我原来心目中的日本人素质稍有反差。不过绝大多数人都是不会这样的，我见的是极个别现象。

日本的女的大多数是家庭妇女，这个传统没变。所以平日的白天，要在中国正是超市人最少的时候，在日本相反，比平时晚上人多多了，都是没工作的家庭主妇去购物的。日本人没有像中国那样推个大车子，堆了一车东西的。这和日本遍地都是超市有关。

日本的就业主要是靠大企业为龙头，带动下游各个小的配件企业，有了这些广大的上班一族后，带动了巨大的服务业。中国还处于工业化都一定程度，城市化刚刚开始，服务业还没有被发展起来的阶段，这就未来的中国模式。

等到中国经济发展到一定程度，产品有一定的竞争力后，就要挤兑日本的企业，那时候中国人要和日本人来抢资源了。真到了那一天，亚太经济中心从日本转移到长 三角，日本的大企业会受到挤压，那时候日本的社会的稳定性将受到致命打击。这也是命，也是命中注定，命该如此，怪就怪日本国土太小。没办法阻挡历史的车 轮。不久的将来，真的中国人就会实现梦回唐朝的愿望了。

语言有障碍，工作上很是被动啊。日本人要我来是利用廉价劳动力来干活的，可不是用他们的人工来陪我练口语的。

在日本第八天

发表于 2011-09-05   |   分类于 未分类   |  

来了日本这些人，所看到的所感受到的很以前在网络上，电视里，别人说的很是不一样。有偏见，有夸张，有不准确，有绝对化的成分。

也许很多来过日本的人是那些留学生，是那些边打工边学习的，也许是忙于穿梭在东京，大阪这种大城市里，也许他们根本就是来旅游一番，没有在日本生活下来。

日本人的生活节奏也不快啊，走路就没看到带小跑的，以前听这么说也许是因为人家在大的交通换乘赶车吧。

说日本开车难，开车不方便，总是堵车。我住的地方离大阪差不多10公里，没见过一次堵车，且顺畅的很，车速比国内城市开车要快多了。国内上班高峰不堵车就怪了，日本那是上班高峰堵车是绝对不可能的。但是他们上班开车的几乎没有，主要还是停车太难了，没有地方可以停车。

我在日本这么就，除了看到一个开保时捷的傻B把车停在路边外，我一次都没有看到日本人把车停在路边，或紧靠路边停或停在人行道上。所以他们交通顺畅，根本 就没有堵的可能性。当然，还和行人，自行车的自觉有关，没有人会不在人行横道过马路。所有自行车，行人都自觉在人行横道处等红灯，然后通过，不会有人从路 中间过马路，虽然日本的马路没有任何隔离措施。所以车速快嘛，不会因为有行人会踩刹车。

日本停车难是真的，很少 的停车位。在大阪的梅田JR站，看到一处停车场收费是500日元一小时。

奇怪了，在国内的公司中央空调声音很大，关掉一下子整个环境都安静了，但在日本公司的办公环境，根本听不见空调的声音。果真是他们的电器和我们的标准就不一样。

吸尘器也是如此，国内的声音巨大无比，日本的声音很小。

日本的卫生纸有点不习惯，都是很薄很薄的一层，国内卖的都是三层的。

日本人的厕所也觉得怪怪的，他们往往都是电子化的。还帮你除臭，洗屁股，烘干。冲水的时候是先吸掉，再冲，这样会保证冲过后，水是清的。

在日本第七天

发表于 2011-09-04   |   分类于 未分类   |  

去吃日本第一次放题。

日式油炸。

算是比较贵的一种放题了。

中国人吃比较合算，因为中国人的胃口不知道比日本人要大多少。

下次去吃旁边的日式火锅放题和西餐放题。

2000日元一人。还是相当合算的，这个价格在日本。

我们基本都是吃的油炸虾。外加一些点心和油炸牛肉。

里面的冰淇淋很好吃。可惜没有时间了，限定70分钟。吃了一个不得不走了。

日本放题很多都是限定时间的。所以是后给钱，防止超时，好算钱。

本以为日本人吃放题时很文雅，但发现日本人也是拼命吃，只是他们胃口小，想多吃也吃不下去啊。

日本开车太礼貌 了，总是让这你，你想让他先走，他偏不走，大老远的就停在那里等你走，你不走就僵持在那里。唉。太文明了，有时候也别扭。

去了日本的书店，感觉日本的书不贵，纸贵。

去看小孩看的书，发现自己一下子竟然看不懂。因为小孩看的书都是假名，对日本人来说，乃至对外国人呢来说，假名容易，汉字是高级阶段。发现我暂时还是适合阅读成人的东西。

日本的电话和中国不太一样。拿起电话不是连续的嘟，而是如中国的占线的嘟嘟嘟的短促的声音。

日本的超市是多，沿着路走个十几分钟能路过二十几个超市。且日本的购物环境特别好，没个店都不会像欧尚那么大，但是都是很温馨。

以前都说日本人走路都带小跑的，至少在伊丹这里没看到这种景象，日本人还是很悠闲的在生活。

日本大多数人都是很讲文明的。但偶尔还是有些素质低下的。所以我过马路还是要左右看看的。按理在日本只要是绿灯，就可以放心的过马路，所有车子都会让的。 但是就怕也许万分之一的不讲素质的人给碰上的。但目前我还是没有遇到过。但目前我唯一发现一次拐弯车辆没有让直行车辆的情景。

我在国内肯定是高素质的人，但在日本绝对是低素质的人。因为有些实在是做不到日本人的要求。比如喝过的牛奶纸盒。按理都要剪开，冲洗干净，再晒干，然后分类扔掉。我实在没心思弄，喝完了就放到垃圾堆里。整理垃圾的人一看到这个自然会想到这袋垃圾是中国人呢扔的。

也难怪日本人看不起中国人，确实住在一起的日本人很少给别人添麻烦，只有中国人在楼道里大声喧哗，洗澡大声喧哗，上完厕所，洗过衣服不关灯，拖鞋总是反方向放，不正放，往纸篓里扔卫生纸（日本纸篓只是给你扔卫生纸外面包装的塑料质的）。

不过有时候日本人也有双重标准，有些地方偏袒自己人，看不惯中国人，不一视同仁。

在日本第六天

发表于 2011-09-03   |   分类于 未分类   |  

日本成年女的基本都是化妆的。

日本女的也很辛苦。在中国，女的打扮打扮，化化妆就会很突出，吸引别人注意。日本女的辛辛苦苦的打扮，化妆也只能落得和其他女的一样，会淹没在众人之中，但是要是你不打扮，那就很另类，别人会觉得很滑稽，会异样的看着你。

日本就是路边的小的快餐店也自动化，买套餐或点菜也在自动贩卖机上操作，然后把票拿给服务员。

把日本的人工和农产品的价格考虑进去，日本吃的真的超级便宜。

日本的电视大半都是高清的，且可以选择有没有字幕，字幕是单独放送的。看看电视，语言提高很快啊。

全天受台风影响，狂风暴雨，下了一整天。中国最近很多大城市淹水，那下的雨和今天下 的雨那真是小巫见大巫了，要在中国下这么大的雨，早不知道淹了几层楼了。但在日本，一定没有这个担心，因为知道这个对日本来说那是家常便饭。

电视里不停得播放字幕，各地的天气情况，避难情况等等。

在日本第五天

发表于 2011-09-02   |   分类于 未分类   |  

去了大阪车站。伊丹到大阪做JR是到大阪梅田站。

15分钟就可以到大阪。伊丹离大阪是近，可以说比苏州的园区到新区还要近。可以伊丹和大阪还不是紧挨的城市，可见日本的所谓市有多小了。去中国市里的区小多了。

到了大阪第一次感受到了日本的人多。当然，要是上班的时候去，人更多。

一到大阪，才感受到了日本的高楼大厦。

日本人非常自觉，乘电梯都是自觉在站在电梯的右边，左边给那些走电梯的人用。这不仅在大阪这样，在日本都是这样。

去车站附近问询处问了下，找到了旅游管理处，本想买几张关西三日券。结果被告知不能做JR，于是就不买了。

出了大阪车站就看到了一个建在屋顶的摩天轮。本想去坐的，可是出去后就找不到了。

走在大阪的行人道上，就是感觉很其他地方的一样，很窄，但是大阪的行人道不能停自行车，大阪有相当的范围内是不能停自行车的，骑了经过可以，但是不能停。这个在中国是没有的。

去了大阪的一家路易斯威登，很不自在，营业员上来就问can I help you。一眼就看出我们是外国人。

大阪的yodobashi camela和中国的电器卖场差不多。里面卖的产品，中国也基本都有，价格甚至比日本的便宜。现在来日本逛数码店，买数码产品已经没什么优势了。

我又不买单反，不然至少这里品种要比国内多得多。

来日本买东西一定要看仔细了，一定要是日本制造。悲剧的是买了中国制造，大老远带回去，才发现兜了个大圈子。

其实日本的大多数商品都是中国制造的，特别是衣服，电器，器皿，塑料材料等等。

日本没有网络限制。国内的所谓的危害国家安全的信息都可以访问。就是有一处非常不爽，土豆，优酷，qiyi等国内大的视频网站都因版权问题不提供大陆以外的地区观看。

日本再怎么好。在享受盗版的便利和廉价的人力方面远远不及中国啊，远远没有中国好。

有个地方欠考虑，原以为路由器上可以挂很多台电脑。但实践下来，5挂了5台电脑，路由严重超负荷，根本不能保证5台电脑的上网。经常因超负荷而死机。不止我这一台，所有的路由都是如此。

第一天上班。

出了寮，管理员都会说一路走好。我们也要礼貌说：我走了。

公司在公园的旁边，环境很好。

走的一路上所有车子都会让行人，要么减速很慢，要么减到停下来。

写字楼也很好。

大门的保安每见一个人都要说你好。所有进们的时候不仅要挂胸卡，且要把胸卡送前一点给保安看仔细。

公司环境很好，见了科长。

给了礼物。

设定了邮件，把准备的资料都复制到本地电脑。

笔记本很新，很强大，就是屏幕太小了。

上午听了个听了好几遍的讲座。

中午饭，都说日本饭少，吃不饱。我感觉还好，基本能饱。

下午昼礼要自我介绍，我一点准备都没有。简单说了下。

这帮一起来的人就整天聊天，我无语了，大好机会浪费掉了。

日本的厕所电子设备真多。感应的东西多，这点国内好的厕所也有。但不同的地方每个水池边上都有布，不是插手用的，而是给你擦溅出来的水用的。大便用的也是电子的，可以冲洗屁股和烘干的。

日本不用担心没水喝，随便一个龙头拧开就可以直接喝。路边也到处都是自动售货机，各种饮料。

晚饭在寮里吃的，日本人要求饭前必须洗手。吃饭都要和做饭的人打招呼，晚饭吃得很饱，也比较丰富。

孙双比我早去一个星期，对周边就已经很熟悉了，晚上还带我出去。且都能和日本人作简单的交流了，学习适应力也太强了吧。

我背着包，穿着圆领T恤，路上不少人都盯着我看，原来日本人很少穿圆领T恤的，且很少人背包。显得我很突出，扎眼，以后在日本不能这么穿了。

去了日本的百元店，里面发现些宝贝，金属的调羹和刀叉很便宜，我买了一套。

回来后洗了澡。泡了泡澡堂。日本的澡堂清澈见底，中国绝对做不到，因为日本人都很自觉的先洗干净身子再下去泡泡。

在日本吃得好，住的好，什么好，且超市购物还这么便宜，衣服也很便宜，生活成本这么低，搞得我都不想回国了。

不过也有不文明的日本年轻人骑摩托车飚车的。

日本的人行道和自行车道是共用的。很窄。且是柏油的，不像中国铺了砖头，这一点远不及中国。

路过中介店，拿了些日本房屋租赁和出售的信息，原来日本租房和买房是如此的便宜。颠覆了我的想象。

太困了，回去后倒头就睡了。
