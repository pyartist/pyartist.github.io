在日本第三十天

发表于 2011-09-27   |   分类于 未分类   |  

日本的道路

国道就单向2车道。

省道就单向1车道。

但利用率比中国有的城市的双向10车道也许都要高。

反正我在日本这么就没看到堵车过。

日本行车效率高主要是他们几乎没有人乱停车在次干道上，更不用说主干道了，还有没有人超车。所有车都保持一个速度，且是比较快的速度，日本人在繁忙的道路 上行人一定遵守红绿灯，不忙的路上红灯的时候也要确认来车后才过。日本的开车的人速度都很快，就把所有人不会乱过马路为前提，在中国，看到有行人紧贴机动 车道要过马路了，车子都会减速的，在日本可不会。

日本加油站的油要比中国贵一截。估计都是花在人工上，因为日本的加油站一个加油的时薪大概在1400日元，就是一天干8小时，能拿1千人民币的日薪。

日本加油站都很小，有的加油站为了省面积，是从上往下加的，就是输油管道是在上面，电子显示屏也在上面，地面就是一大块平地。还有一种省面积的方式是，所有的显示屏都集中在一个地方，对好自己的加油的号码，自己去看对应号码的显示屏。

日本加油站给车子加油的时候，在加油过程中，会替你把窗子擦一擦。

去了西河源公园。

去了茨木市的中央公园。

去了川端康成博物馆。

回去的时候路过茨木神社，进去看了下，穿过阪急本通商店街到了阪急茨木市站。

在日本第二十六天

发表于 2011-09-23   |   分类于 未分类   |  
日本一个小公园往往连中国的一个小区的空地都比不上，一个小区的绿化，一个小区的健身设施都比不上。我说的只是面积比不上。
日本的操场和中国的学校不同，几乎没有用塑胶的，但利用率非常高，总看到有占满整个操场的人在活动。不像中国只有白天上体育课的时候才看到利用，日本几乎每一个操场都能看到的运动就是棒球。
日本人的礼貌从小做起，见到你了都要打声招呼，不管认识不认识。路过小学门口的时候，总有老师和小学生站在校门口给过路的人打招呼。
日本人汉字功底不是一般的差，写汉字是歪歪扭扭，笔画都写不全。估计是日本人笨，不仅发音发不出几个音，连手腕也不灵活，写字歪歪扭扭，看他们的字太痛苦。
日本发达的排水系统。所有的道路最边上都有排水系统，一般都是直接裸露在外面，支流汇到干流，一级级汇上去，直到大海，且他们的排水渠都非常深，一般暴雨估计才用到四分之一，三分一，还有很多的余量，日本人是为了防止更大的台风暴雨袭击的，多少年一遇的那种。

在日本第二十二天

发表于 2011-09-19   |   分类于 未分类   |  

从伊丹坐电车到神户花时不到1小时。

到了阪急三宫站。

算个大站。也是大的中转站，和JR基本连在一起。还有阪神高速，还有神户的地铁也连在一起。

出了站一直往南走。

去了神户的政府大楼。他们的政府大楼可以供市民使用的。

二楼是展厅，24楼是观光层，其他的层是政府税务，财政等用于办公的。

走进电梯，电梯门关上，电梯小姐来了一个90度鞠躬。

先去二楼看了市民画展，后去24层，可以俯瞰整个神户港。

看到神户的宣传，10月上旬是旅游周，每一天都有对应的场馆搞活动，这不很明显针对即将到来了中国黄金周嘛。

接下来去了东游园地公园，附近有个花计时，有阪神地震纪念的碑文，当时保留下来石柱，有个为了镇魂和慰灵的希望之灯。回来的时候看到灯下悄悄地多了束鲜花。

东游园地公园以前只对外国人开放，也是日本学习西方文化在重要窗口。马拉松，西装，保龄球就是从这里传向全日本。

路过了一些欧美风格的建筑。神户的移民挺多的。也保留了些各地的建筑风格。

去了神户的海边走了走。中国游人特别多，然后中国人拖了一车一车类似中国小商品市场的垃圾货放在外面卖来忽悠来这的中国人。

去了神户的中华街——南京街，人挤人，中国大陆人，台湾人很多。

去的时候有人在敲锣打鼓，舞狮子。

基本都是吃的店铺，中华最得意的也是吃。

其中看到一家中医门诊，外面的人都排起队来了，看来中医还是有一定市场的。

去须磨海滨公园。天气有点凉，沙滩上没什么人。

去了须磨浦公园。

去了舞子。

到了舞子。体会到了日本人为什么说“东京起步，大阪挣钱，定居神户”。屋子风景宜人，交通方便，也没那么拥挤，城市建设的也挺好。

去了明石公园。看到了明石海峡大桥。跨度够大，汽车，电车分层的。我长这么大，第一次同时看到轮船，汽车，火车（电车），飞机这些交通工具同时出现在眼前的奇景。

桥看上去都是钢结构，还看到有人在桥中间吃饭，桥上架了个餐厅。

明石公园很大，环境很好，也安静。看到了孙文的移情阁。

去了屋子的海滩，迎着落日看着明石海峡大桥，吹着海风，视觉，听觉，触觉，嗅觉的多重享受，周边很多老人公寓，这个环境太适合养生了。

走走就到了垂水了，去了垂水的奥特拉斯。东西超级便宜。中国的高档品在这里就是白菜价。买了一双耐克的鞋子，一条耐克的裤子，都是1百多人名币，买了2个耐克的包。

去了李维斯，买了4件衣服，价格都在一百多元人名币，国内李维斯的裤子一般都要在一千元以上。

中国人挣钱少，衣服却贵的离谱。

在日本第二十天

发表于 2011-09-17   |   分类于 未分类   |  

日本基本都是山地，主要在沿海部分有些平地，还都是人工给弄平的。

今天去了甲山一带。甲殤园一带。

到了甲殤园车站，下了电车就看到有一排的各地导游图。
是为了方便步行者游览的短距离线路图和沿线景点，一搬的步行距离在3～9公里。

从车站走去甲山，一路都是上坡，其实整个城市是建在山上，屋子也建在山上，高高低低的。日本人真是利用尽了每一寸能利用的土地。

去爬了甲山，爬到山顶海拔才300多米，爬的地方就200多米，一共才爬了100米左右。

山道人工铺的，和国内差不多。

山顶被削平了，估计有特殊用途。

下了甲山，有个人造湿地。日本人什么都是微缩的，竟然人造一个生态湿地，用于小孩的环境教育。

下面去了甲山森林公园。环境不错，最大的特色就是爱的像，新谷秀夫雕刻的作品。

甲山竟然看到有好几处提示牌，提醒小心野猪出没。晕，日本的环境也忒原始了点吧。

路上路过自然之家，都是大人陪小孩来体验的，我就没进去了。

甲山有几处可以俯瞰三宫市，俯瞰大阪市的地方。我去了一处，可以俯瞰大阪和三宫市，一直到海。远处的电车就像一个毛毛虫在缓缓的爬。

据说夜景很美。我没有呆到那么晚，下午就回去了。

在日本第十五天

发表于 2011-09-12   |   分类于 未分类   |  

在下班的路上，一起走的同事，叫了一声蜂鸟。我都没有留意。会过去仔细看了看。彩色的，偏嫩黄色的羽毛，尾部是黑的，头有个如空中加油机般的长长的管子，逐一在花朵上吸东西。动作挺快的，如直升飞机般。回来上网查了下，果然刚刚看的是蜂鸟。世界上最小的蜂鸟，心跳上百次。

下班的路上看到一辆在卖的奔驰车，才70多万日元，折合人民币差不多6万元。二手的，用了15年了。是超级便宜。

日本人对中国人和对待中国还是很倾向于善意的。在日本人的眼里，中国是一个问题多多的国家，但是确实在不断进步和改善，为何不多看看中国好的一面呢。日本 对中国如此，还是比较客观的。中国对日本呢，客观上讲是一种利用。感谢日本让中国利用这份情感来保证中国稳定的重要一环。总比文革强，有破坏性，喊喊口 号，不具有破坏性，但一样可以起到共同的敌人就能更加团结。这种转嫁控制得好的话是利国利民，控制不当就不行了。且只能在发展初期，矛盾尖锐时用，以后国 家发展成熟了，可能就不能用了。

日本酒驾，不仅对司机重罚，对坐同一车的其他人员也要处罚，对喝酒的饭店也要处罚。这个非常合理。同车的人有没有承担劝阻的责任。饭店有没有劝阻的责任。 这样可以极大的减少酒驾，比单纯提高对本人的罚金和惩罚力度要效果更好，当然不是说日本对本人处罚轻。本人要承担50万也不知道是100万的罚金。就是一 辆车的钱被罚没了。

日本机动车是靠左行驶的，行人是靠右行驶的，自行车是靠左行驶的。主要日本地方太小了，自行车和行人共用一个很窄的道，为了互相避让，不得不这样。

日本在路上行驶的汽车方向盘一般在右边，但也允许在左边，比如进口汽车。反正在中国我是没看见路上有开的车方向盘在右边的。

现在有了几把的硬币了，不知道以后怎么用得掉，日本最大钞票是1万，商品往往都是精确到1元的，就如在中国价格标到分，那个零钱那个多啊。

在日本第十四天

发表于 2011-09-11   |   分类于 未分类   |  

中午吃了火锅放题，品种很少，很多限制，肉都是要服务员送上来的，半放题性质，也被日本人给忽悠了，他们的蘸料就一种，食材又少得可怜。更荒唐的是，日本人火锅的汤底竟然是白开水，难道日式火锅就是如此吗！晕。

今天去了sky part，就是机场边的公园。机场公园有陈列室，影音介绍。有很多儿童娱乐的设施。因为去这个公园的大多是家长带着小孩去的。

这个公园主要的目的是提供一个最近距离观看飞机起飞降落的观景台，其次起噪音缓冲带的作用。

现在飞机发动机是牛叉，声音很小了。有些小飞机就是到附近听到点声音，几秒后就基本听不到声音起飞了。

很多人做过无数次飞机，但可能没好好看到过飞机降落起飞的全过程。

大阪国际机场不算个大机场，可是飞机不断，两～五分钟就有一架飞机降落起飞。这次是看遍了大型机，小型机，直升机的起飞降落。

日本空气虽干净，能见度也高，却带来了负面的效果，就是太阳直射太强烈了。

今天去仔细看了看日本的肉价，菜价，水果价格。总体感受就是相对日本人的收入，他们的这些生活必需品真是太便宜了。肉有的比国内还便宜，只有那种雪花牛肉 才贵，其他都不贵。水果都是论个买的，但也还好，国内好的水果也要卖接近这么贵。蔬菜比国内贵些，但也不是很离谱，反正日本人吃得少，但是蔬菜的品种实在 是少的可怜。今天吃自助日式火锅，蔬菜也少得可怜。日本人在吃的食材上挺可怜的，不然他们也不会在菜的色上下那么多功夫。他们下那么多的功夫就是为了来弥 补食材上的不足。

中华料理在日本遍地都是，且往往是身份的象征，都很贵的。见过离奇的一碗蛋炒饭买8000日元，就是700人名币一碗。晕死。日本人太好忽悠了。可见日本人对外来的好的东西很是尊重。不会像中国一样讨厌一个东西就否定其一切。

在日本第十三天

发表于 2011-09-10   |   分类于 未分类   |  

去了昆扬池公园。是伊丹最大的公园和景点了。

和以前去过的公园最大的不同就是遍地是鸟，湖边，书上，栏杆上到处是成群的鸟，都是体积很大的鸟，这种鸟在国内看到一只都会惊喜万分，这里到处都是且根本就不怕人的，走过去，他们根本就没把你当回事。日本人真正做到了人和鸟的和谐相处。

水质很好，水里有很多游的叫不出来的东东，估计叫他们鸭子吧。还有大海龟在水中游。

有三个地方给我印象挺深的。

这里隔200米就有个洗手的地方，原来是怕人接触到了鸟类的粪便，带了病菌。

不允许往湖里乱扔东西，破坏环境，否则以危害公共安全罪来惩罚。

分析可能存在的水的污染源，（湖底淤泥，下水道的水渗透进来，鸟的粪便），并逐一想办法对策。

其实日本对醉驾惩罚比中国重多了。罚几十万也不知道上百万，还有坐半年劳。不服从交警指示，驾车逃跑，也要做几个月的劳，交警可以佩枪的。哪像中国可以骂交警，还动手。
